# stringifyJson

JSON Objects stringifier

```typescript
stringifyJson = (object: unknown) => string
```  

## Parameters
|          |            | type     | default value
| :-:      | :--        | :--      | :--           
| required | **object** | `unknown` |

## Returns
`string`

## Throws
Error when the given `object` cannot be `JSON.stringify`d
