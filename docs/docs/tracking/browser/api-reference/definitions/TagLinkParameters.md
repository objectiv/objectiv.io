# TagLinkParameters

[tagLink](/tracking/browser/api-reference/locationTaggers/tagLink.md) has an extra attributes, `href`, as mandatory parameter.

```typescript
type TagLinkParameters = LocationTaggerParameters & { 
  href: string
};
```

|          |          | type   | default value
| :-:      | :--      | :--    | :--           
| required | **href** | string |
