# useLocationStack

Consumes [LocationProviderContext](/tracking/react/api-reference/providers/LocationProviderContext.md) and retrieves its `locationStack` property.

```ts
useLocationStack = () => LocationStack
```

## Parameters
None.

## Returns
`LocationStack`
