# useTracker

Consumes [TrackerProviderContext](/tracking/react/api-reference/providers/TrackerProviderContext.md) and retrieves its `tracker` property.

```ts
useTracker = () => ReactTracker
```

## Parameters
None.

## Returns
`ReactTracker`
