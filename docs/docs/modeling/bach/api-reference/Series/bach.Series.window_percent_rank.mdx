# bach.Series.window_percent_rank

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">window_percent_rank(</span><small class="parameter-list"><span class="parameter" id="window">window=None</span></small>)</h2>


<div class="content">

Returns the relative rank of the current row, that is
(rank - 1) / (total partition rows - 1).
The value thus ranges from 0 to 1 inclusive.
