# bach.SeriesAbstractNumeric.scale

<div class="method">
<h2 class="signature-class"><span class="additional-name">SeriesAbstractNumeric.</span><span class="name">scale(</span><small class="parameter-list"><span class="parameter" id="with_mean">with_mean=True</span>, <span class="parameter" id="with_std">with_std=True</span></small>)</h2>


<div class="content">

Standardizes series based on mean and population standard deviation.


### Parameters

* **`with_mean`** (_bool) – if true, each value will be centered before scaling

* **`with_std`** (_bool) – if true, each value will be scaled to unit variance



### Returns

Series



### Return type

[bach.series.series.Series](/docs/modeling/bach/api-reference/Series/bach.Series/#bach.Series)
