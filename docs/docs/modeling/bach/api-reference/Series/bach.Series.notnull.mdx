# bach.Series.notnull

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">notnull(</span><small class="parameter-list"></small>)</h2>


<div class="content">

Evaluate for every row in this series whether the value is not missing or NULL.

**NOTE**: Only NULL values in the Series in the underlying sql table will return True. numpy.nan is not
checked for.
