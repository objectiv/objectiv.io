# bach.Series.any_value

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">any_value(</span><small class="parameter-list"></small>)</h2>


<div class="content">

Example: a > b.any() evaluates to True is a > b for any value of b.
