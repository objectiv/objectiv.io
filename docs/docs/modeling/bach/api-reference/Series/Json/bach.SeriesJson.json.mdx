# bach.SeriesJson.json

<div className="property">

: bach.series.series_json.JsonAccessor

Get access to json operations via the class that’s returned through this accessor.
Use as `my_series.json.get_value()` or `my_series.json[:2]`

<div className="method">

### <span className="name">array_contains</span>

<small className="parameter-list">(<span className="parameter" id="item">item</span>)</small>


<div className="content">

Find if an object or list of objects are contained in the array.


#### Parameters

**`item`** – items to be verified if it is contained by the array.



#### Returns

boolean series indicating if the array contains all elements.


This assumes the top-level item in the json is an array. Will result in an exception (later on) if
that’s not the case!


</div>


</div>

<small className="parameter-list">()</small>


<div className="content">

Get the length of the toplevel array.

This assumes the top-level item in the json is an array. Will result in an exception (later on) if
that’s not the case!


</div>


</div>

<div className="method">

### <span className="name">get_value</span>

<small className="parameter-list">(<span className="parameter" id="key">key</span>, <span className="parameter" id="as_str">as_str=False</span>)</small>


<div className="content">

Get item from toplevel object by key.


#### Parameters

* **`key`** – the key to return the values for.
* **`as_str`** – if True, it returns a string Series, json otherwise. Particular useful if the returned
value is in fact a string. In json the string will be represented as a quoted string, if this
field is set to true, the returned SeriesString will not have additional quotes.



#### Returns

series with the selected object value.



</div>
