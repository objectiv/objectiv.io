### <span className="name">sql_format</span>

<small className="parameter-list">(<span className="parameter" id="format_str">format_str</span>)</small>


<div className="content">

Allow formatting of this Series (to a string type).


#### Parameters

**`format_str`** – The format to apply to the date/time column.
Currently, this uses Postgres’ data format string syntax:
[https://www.postgresql.org/docs/14/functions-formatting.html](https://www.postgresql.org/docs/14/functions-formatting.html)


:::warning

This method is deprecated, we recommend using `SeriesAbstractDateTime.dt.strftime()` instead.


:::

```python
df['year'] = df.some_date_series.dt.sql_format('YYYY')  # return year
df['date'] = df.some_date_series.dt.sql_format('YYYYMMDD')  # return date
```


#### Returns

a SeriesString containing the formatted date.

### <span className="name">strftime</span>

<small className="parameter-list">(<span className="parameter" id="format_str">format_str</span>)</small>


<div className="content">

Allow formatting of this Series (to a string type).


#### Parameters

**`format_str`** – The format to apply to the date/time column.
This uses  1989 C standard format codes:
[https://docs.python.org/3/library/datetime.html#strftime-and-strptime-format-codes](https://docs.python.org/3/library/datetime.html#strftime-and-strptime-format-codes)


```python
df['year'] = df.some_date_series.dt.sql_format('%Y')  # return year
df['date'] = df.some_date_series.dt.sql_format('%Y%m%d')  # return date
```


#### Returns

a SeriesString containing the formatted date.
