# bach.Series.value_counts

<div class="method">

<h2 class="signature-method"><span class="additional-name">Series.</span><span class="name">value_counts(</span><small class="parameter-list"><span class="parameter" id="normalize">normalize=False</span>, <span class="parameter" id="sort">sort=True</span>, <span class="parameter" id="ascending">ascending=False</span>, <span class="parameter" id="bins">bins=None</span>, <span class="parameter" id="method">method='pandas'</span></small>)</h2>


&#8203;<span class="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/series/series.py)</span>

<div class="content">

Returns a series containing counts per unique value


### Parameters

* **`normalize`** (bool) – returns proportions instead of frequencies

* **`sort`** (bool) – sorts result by frequencies

* **`ascending`** (bool) – sorts values in ascending order if true.

* **`bins`** (Optional[int]) – works only with numeric series, groups values into the request amount of bins
and counts values based on each range.

* **`method`** (str) – Method to use for calculating bin ranges.
Supported values:

>   * ”pandas” (default): Performs bound adjustments based on Pandas implementation.

>   * ”bach”: No bound adjustments are performed. Instead, first interval includes both
> lower and upper bounds.




### Returns

a series containing all counts per unique row.

[bach.series.series.Series](/docs/modeling/bach/api-reference/Series/bach.Series)
