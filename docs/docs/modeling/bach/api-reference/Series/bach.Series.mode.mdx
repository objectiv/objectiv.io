# bach.Series.mode

<div className="method">

<h2 className="signature-method"><span className="additional-name">Series.</span><span className="name">mode(</span><small className="parameter-list"><span className="parameter" id="partition">partition=None</span>, <span className="parameter" id="skipna">skipna=True</span></small>)</h2>

Returns the mode in each partition or for all values if none is given.


### Parameters

* **`partition`** (Union[GroupBy, [DataFrame](/modeling/bach/api-reference/DataFrame/index.mdx)]) – The partition or window to apply


* **`skipna`** (bool) – only `skipna=True` supported. This means NULL values are ignored.





### Returns

a new Series with the aggregation applied
