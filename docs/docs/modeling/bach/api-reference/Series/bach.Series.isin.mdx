# bach.Series.isin

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">isin(</span><small class="parameter-list"><span class="parameter" id="other">other</span></small>)</h2>


<div class="content">

Evaluate for every row in this series whether the value is contained in other

Example: a.isin(b) evaluates to True for a specific row if a > b for all values of b.
