# bach.Series.reset_index

<div className="method">

### <span className="name">reset_index</span>

<small className="parameter-list">(<span className="parameter" id="level">level=None</span>, <span className="parameter" id="drop">drop=False</span>)</small>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/mainon3.8/site-packages/bach/series/series.py)</span>

<div className="content">

Drops the current index.


#### Parameters

* **`level`** (Optional[Union[str, Sequence[str]]]) – Removes given levels from index. Removes all levels by default
* **`drop`** (bool) – if False, the dropped index is added to the data columns of the DataFrame. If True it
is removed.



#### Returns

Series or DataFrame with the index dropped.



#### Return type

*Union*[[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx), [bach.series.series.Series](/modeling/bach/api-reference/Series/bach.Series.mdx)]
