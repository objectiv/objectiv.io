# bach.SeriesString.get_dummies

<div className="method">

### <span className="name">get_dummies</span>

<small className="parameter-list">(<span className="parameter" id="prefix">prefix=None</span>, <span className="parameter" id="prefix_sep">prefix_sep='_'</span>, <span className="parameter" id="dummy_na">dummy_na=False</span>, <span className="parameter" id="dtype">dtype='int64'</span>)</small>

Convert each unique category/value from the series into a dummy/indicator variable.


#### Parameters

* **`prefix`** (Optional[str]) – String to append to each new column name. By default, the prefix will be the name of
the caller.
* **`prefix_sep`** (str) – Separated between the prefix and label.
* **`dummy_na`** (bool) – If true, it will include `nan` as a variable.
* **`dtype`** (str) – dtype of all new columns



#### Returns

DataFrame



#### Return type

[DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)


:::note

Series should contain at least one index level.


:::
