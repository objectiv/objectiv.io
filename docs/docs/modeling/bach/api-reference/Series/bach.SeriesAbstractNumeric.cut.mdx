# bach.SeriesAbstractNumeric.cut

<div class="method">
<h2 class="signature-class"><span class="additional-name">SeriesAbstractNumeric.</span><span class="name">cut(</span><small class="parameter-list"><span class="parameter" id="bins">bins</span>, <span class="parameter" id="right">right=True</span></small>)</h2>


<div class="content">

Segments values into bins.


### Parameters

* **`bins`** (_int) – The amount of bins to segment data into

* **`right`** (_bool) – If true (by default), each bin will include the rightmost edge. (e.g (x,y]).
