# bach.Series.dtype

<div>

<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">dtype(</span><span class="type-annotation"><em>: st</em></span> <span class="type-annotation"><em> = '</em></span> )</h2>


<div class="content">

The dtype of this Series. Must be overridden by subclasses.

The dtype is used to uniquely identify data of the type that is
represented by this Series subclass. The dtype must be unique among all Series subclasses.
