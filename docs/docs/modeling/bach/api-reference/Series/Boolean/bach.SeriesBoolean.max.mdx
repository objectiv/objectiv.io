# bach.SeriesBoolean.max

<div className="method">

### <span className="name">max</span>

<small className="parameter-list">(<span className="parameter" id="partition">partition=None</span>, <span className="parameter" id="skipna">skipna=True</span>)</small>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/series/series_boolean.py)</span>

<div className="content">

Returns the maximum value in the partition, i.e. value will be False iff all values in the
partition are False.


#### Parameters

* **`partition`** (Union[GroupBy, [DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)]) – The partition or window to apply
* **`skipna`** (bool) – only `skipna=True` supported. This means NULL values are ignored.



#### Returns

a new Series with the aggregation applied
