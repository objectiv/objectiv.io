# bach.Series.window_cume_dist

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">window_cume_dist(</span><small class="parameter-list"><span class="parameter" id="window">window=None</span></small>)</h2>


<div class="content">

Returns the cumulative distribution, that is
(number of partition rows preceding or peers with current row) / (total partition rows).
The value thus ranges from 1/N to 1.
