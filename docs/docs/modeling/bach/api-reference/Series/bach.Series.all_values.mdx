# bach.Series.all_values

<div class="method">
<h2 class="signature-class"><span class="additional-name">Series.</span><span class="name">all_values(</span><small class="parameter-list"></small>)</h2>


<div class="content">

Example: a > b.all() evaluates to True is a > b for all values of b.
