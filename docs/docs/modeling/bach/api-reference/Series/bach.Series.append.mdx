Append rows of other series to the caller series.


#### Parameters

* **`other`** (Union[[bach.series.series.Series](/modeling/bach/api-reference/Series/bach.Series.mdx), List[[bach.series.series.Series](/modeling/bach/api-reference/Series/bach.Series.mdx)]]) – objects to be added
* **`ignore_index`** (bool) – if true, drops indexes of all objects to be appended



#### Returns
