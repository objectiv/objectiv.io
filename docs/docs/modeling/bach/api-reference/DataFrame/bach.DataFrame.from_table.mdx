# bach.DataFrame.from_table

<div class="method">
<h2 class="signature-class"><span class="type-annotation"><em>classmethod</em></span> <span class="additional-name">DataFrame.</span><span class="name">from_table(</span><small class="parameter-list"><span class="parameter" id="engine">engine</span>, <span class="parameter" id="table_name">table_name</span>, <span class="parameter" id="index">index</span>, <span class="parameter" id="all_dtypes">all_dtypes=None</span></small>)</h2>


<div class="content">

Instantiate a new DataFrame based on the content of an existing table in the database.

If all_dtypes is not specified, the column dtypes are queried from the database’s information
schema.


### Parameters

* **`engine`** (_sqlalchemy.engine.base.Engine) – an sqlalchemy engine for the database.

* **`table_name`** (_str) – the table name that contains the data to instantiate as DataFrame.

* **`index`** (_List_[_str_]) – list of column names that make up the index. At least one column needs to be
selected for the index.

* **`all_dtypes`** (_Optional_[_Dict_[_str_, _str_]_]) – Optional. Mapping from column name to dtype.
Must contain all index and data columns.
Must be in same order as the columns appear in the the sql-model.



### Returns

A DataFrame based on a sql table.



### Return type

[bach.dataframe.DataFrame](/docs/modeling/bach/api-reference/DataFrame/bach.DataFrame/#bach.DataFrame)


**NOTE**: If all_dtypes is not set, then this will query the database.
