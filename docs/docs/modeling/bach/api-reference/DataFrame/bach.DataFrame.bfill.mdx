# bach.DataFrame.bfill

<div className="method">

<h2 className="signature-method"><span className="additional-name">DataFrame.</span><span className="name">bfill(</span><small className="parameter-list"><span className="parameter" id="sort_by">sort_by=None</span>, <span className="parameter" id="ascending">ascending=True</span></small>)</h2>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div className="content">

Fill missing values by using the next non-nullable value in each series.


### Parameters

* **`sort_by`** (Optional[Union[str, Sequence[str]]]) – series label or sequence of labels used to sort values.
Sorting of values is needed since result might be non-deterministic, as rows with NULLs might
yield different results affecting the values to be propagated when using a filling method.


* **`ascending`** (Union[bool, List[bool]]) – Whether to sort ascending (True) or descending (False). If this is a list, then the
`sort_by` must also be a list and `len(ascending) == len(sort_by)`.





### Returns

a new dataframe with filled missing values.



### Return type

[bach.dataframe.DataFrame](/docs/modeling/bach/api-reference/DataFrame/bach.DataFrame.bfill/)


:::note

`sort_by` is required if DataFrame has no order_by.


:::

:::warning

If `sort_by` is non-deterministic, this operation might yield different results after
performing other operations over the resultant dataframe.


:::
