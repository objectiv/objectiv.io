# bach.DataFrame.mode

<div class="method">
<h2 class="signature-class"><span class="additional-name">DataFrame.</span><span class="name">mode(</span><small class="parameter-list"><span class="parameter" id="axis">axis=1</span>, <span class="parameter" id="skipna">skipna=True</span>, <span class="parameter" id="level">level=None</span>, <span class="parameter" id="numeric_only">numeric_only=False</span>, <span class="parameter" id="**">\*\*kwargs</span></small>)</h2>


<div class="content">

Returns the mode of all values in each column.


### Parameters

* **`axis`** – only `axis=1` is supported. This means columns are aggregated.

* **`skipna`** – only `skipna=True` supported. This means NULL values are ignored.

* **`level`** – not supported.

* **`numeric_only`** – whether to aggregate numeric series only, or attempt all.



### Returns

a new DataFrame with the aggregation applied to all selected columns.
