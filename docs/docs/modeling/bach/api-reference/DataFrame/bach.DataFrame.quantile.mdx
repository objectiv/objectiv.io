# bach.DataFrame.quantile

<div class="method">

<h2 class="signature-class"><span class="additional-name">DataFrame.</span><span class="name">quantile(</span><small class="parameter-list"><span class="parameter" id="q">q=0.5</span>, <span class="parameter" id="axis">axis=1</span>, <span class="parameter" id="**">\*\*kwargs</span></small>)</h2>


&#8203;<span class="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div class="content">

Returns the quantile per numeric/timedelta column.


### Parameters

* **`q`** (Union[float, List[float]]) – value or list of values between 0 and 1.

* **`axis`** – only `axis=1` is supported. This means columns are aggregated.



### Returns

a new DataFrame with the aggregation applied to all selected columns.
