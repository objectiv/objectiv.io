# bach.DataFrame.scale

<div className="method">

### <span className="name">scale</span>

<small className="parameter-list">(<span className="parameter" id="with_mean">with_mean=True</span>, <span className="parameter" id="with_std">with_std=True</span>)</small>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div className="content">

Standardizes all numeric series based on mean and population standard deviation.


#### Parameters

* **`with_mean`** (bool) – if true, each feature value will be centered before scaling
* **`with_std`** (bool) – if true, each feature value will be scaled to unit variance



#### Returns

DataFrame



#### Return type

[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)


Each transformation per feature is performed as follows:

```python
>>> feature.to_pandas()
```

```notebook-output
index
a           1
b           2
c           3
d           4
Name: feature, dtype: int64
```

```python
>>> scaled_feature = feature.copy()
>>> if with_mean:
```

```notebook-output
...     scaled_feature -= mean_feature
```

```python
>>> if with_std:
```

```notebook-output
...     scaled_feature /= std_feature
```

```python
>>> scaled_feature.to_pandas()
```

```notebook-output
index
a   -1.341641
b   -0.447214
c    0.447214
d    1.341641
Name: feature, dtype: float64
```

Where:
    * `feature` is the series to be scaled
    * `mean_feature` is the mean of `feature`
    * `std_feature` is the (population-based) standard deviation of `feature`
