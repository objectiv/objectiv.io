# bach.DataFrame.groupby

<div className="method">

<h2 className="signature-method"><span className="additional-name">DataFrame.</span><span className="name">groupby(</span><small className="parameter-list"><span className="parameter" id="by">by=None</span></small>)</h2>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div className="content">

Group by any of the series currently in this DataDrame, both from index as well as data.


### Parameters

**`by`** (Optional[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx), Tuple[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx), Tuple[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx)], ...]], ...], Sequence[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx), List[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx)]], Tuple[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx)], ...]]]]]) – The series to group by. Supported are:

* a string containing a columnn name.


* a series.


* a list of strings or series. A normal group by will be created.


* a list of (strings, series, lists). In this case a grouping list is created.


* a tuple of (strings, series, lists). In this case a grouping set is created.






### Returns

a new DataFrame object with the [`group_by`](/modeling/bach/api-reference/DataFrame/bach.DataFrame.group_by.mdx) attribute set.



### Return type

[DataFrame](/modeling/bach/api-reference/DataFrame/index.mdx)


:::note

If the dataframe is already grouped, we’ll create a grouping list from the initial
grouping combined with this one.


:::
