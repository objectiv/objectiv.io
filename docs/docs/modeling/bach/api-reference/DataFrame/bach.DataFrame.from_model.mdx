# bach.DataFrame.from_model

<div className="method">

### <span className="type-annotation"><em>classmethod</em></span> <span className="name">from_model</span>

<small className="parameter-list">(<span className="parameter" id="engine">engine</span>, <span className="parameter" id="model">model</span>, <span className="parameter" id="index">index</span>, <span className="parameter" id="all_dtypes">all_dtypes=None</span>)</small>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/mainon3.8/site-packages/bach/dataframe.py)</span>

<div className="content">

Instantiate a new DataFrame based on the result of the query defined in `model`.

If all_dtypes is not specified, then a transaction scoped temporary table will be created with
0 result rows from the model. The meta data of this table will be used to deduce the dtypes.


#### Parameters

* **`engine`** (sqlalchemy.engine.base.Engine) – a sqlalchemy engine for the database.
* **`model`** (sql_models.model.SqlModel) – an SqlModel that specifies the queries to instantiate as DataFrame.
* **`index`** (List[str]) – list of column names that make up the index. At least one column needs to be
selected for the index.
* **`all_dtypes`** (Optional[Mapping[str, Union[List[Any], Mapping[str, Any], str]]]) – Optional. Mapping from column name to dtype.
Must contain all index and data columns.
Must be in same order as the columns appear in the the sql-model.



#### Returns

A DataFrame based on an SqlModel



#### Return type

[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)


:::note

If all_dtypes is not set, then this will query the database and create and remove a temporary
table.


:::
