# bach.DataFrame.append

<div className="method">

### <span className="name">append</span>

<small className="parameter-list">(<span className="parameter" id="other">other</span>, <span className="parameter" id="ignore_index">ignore_index=False</span>, <span className="parameter" id="sort">sort=False</span>)</small>


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/mainon3.8/site-packages/bach/dataframe.py)</span>

<div className="content">

Append rows of other dataframes to the the caller dataframe.
Non-shared columns between dataframes are added to the caller.


#### Parameters

* **`other`** (Union[[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx), List[[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)]]) – objects to be added
* **`ignore_index`** (bool) – if true, drops indexes of all object to be appended
* **`sort`** (bool) – if true, columns are sorted alphanumerically



#### Returns

a new dataframe with all rows from appended Dataframes.



#### Return type

[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)
