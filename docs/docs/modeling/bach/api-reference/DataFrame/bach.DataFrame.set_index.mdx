# bach.DataFrame.set_index

<div className="method">

<h2 className="signature-method"><span className="additional-name">DataFrame.</span><span className="name">set_index(</span><small className="parameter-list"><span className="parameter" id="keys">keys</span>, <span className="parameter" id="drop">drop=True</span>, <span className="parameter" id="append">append=False</span></small>)</h2>

<div className="content">

Set this dataframe’s index to the the index given in keys


### Parameters

* **`keys`** (Union[str, [Series](/docs/modeling/bach/api-reference/Series), Sequence[Union[str, [Series](/docs/modeling/bach/api-reference/Series)]]]) – the keys of the new index. Can be a column name str, a Series, or a list of those. If
Series are passed, they should have the same base node as the DataFrame they are set on.


* **`drop`** (bool) – delete columns to be used as the new index.


* **`append`** (bool) – whether to append to the existing index or replace.





### Returns

a DataFrame with the new index.



### Return type

[DataFrame](/docs/modeling/bach/api-reference/DataFrame/bach.DataFrame.set_index/)
