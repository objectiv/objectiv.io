# bach.DataFrame.set_index

<div className="method">

## <span className="additional-name">DataFrame.</span><span className="name">set_index</span>

Set this dataframe’s index to the the index given in keys


### Parameters

* **`keys`** (Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx), Sequence[Union[str, [Series](/modeling/bach/api-reference/Series/index.mdx)]]]) – the keys of the new index. Can be a column name str, a Series, or a list of those. If
Series are passed, they should have the same base node as the DataFrame they are set on.
* **`drop`** (bool) – delete columns to be used as the new index.
* **`append`** (bool) – whether to append to the existing index or replace.



### Returns

a DataFrame with the new index.



### Return type

[DataFrame](/modeling/bach/api-reference/DataFrame/index.mdx)
