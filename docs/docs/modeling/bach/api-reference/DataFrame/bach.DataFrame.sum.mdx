# bach.DataFrame.sum

<div className="method">

## <span className="additional-name">DataFrame.</span><span className="name">sum</span>

<small className="parameter-list">(<span className="parameter" id="axis">axis=1</span>, <span className="parameter" id="skipna">skipna=True</span>, <span className="parameter" id="level">level=None</span>, <span className="parameter" id="numeric_only">numeric_only=False</span>, <span className="parameter" id="min_count">min_count=0</span>, <span className="parameter" id="**">\*\*kwargs</span></small>)

<div className="content">

Returns the sum of all values in each column.


### Parameters

* **`axis`** – only `axis=1` is supported. This means columns are aggregated.
* **`skipna`** – only `skipna=True` supported. This means NULL values are ignored.
* **`level`** – not supported.
* **`numeric_only`** – whether to aggregate numeric series only, or attempt all.
* **`min_count`** – This minimum amount of values (not NULL) to be present before returning a result.



### Returns

a new DataFrame with the aggregation applied to all selected columns.
