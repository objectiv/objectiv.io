# bach.DataFrame.expanding

<div className="method">

## <span className="additional-name">DataFrame.</span><span className="name">expanding</span>

<small className="parameter-list">(<span className="parameter" id="min_periods">min_periods=1</span>, <span className="parameter" id="center">center=False</span></small>)


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div className="content">

Create an expanding window starting with the first row in the group, with at least `min_period`
observations. The result will be right-aligned in the window.

To use grouping as well, first call [`group_by()`](/modeling/bach/api-reference/DataFrame/bach.DataFrame.group_by.mdx) on this frame and call rolling on the result.


### Parameters

* **`min_periods`** (int) – the minimum amount of observations in the window before a value is reported.
* **`center`** (bool) – whether to center the result, currently not supported.
