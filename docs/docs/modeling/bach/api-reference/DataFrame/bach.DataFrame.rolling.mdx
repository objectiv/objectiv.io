# bach.DataFrame.rolling

<div className="method">

## <span className="additional-name">DataFrame.</span><span className="name">rolling</span>

<small className="parameter-list">(<span className="parameter" id="window">window</span>, <span className="parameter" id="min_periods">min_periods=None</span>, <span className="parameter" id="center">center=False</span>, <span className="parameter" id="closed">closed='right'</span></small>)


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/bach/bach/dataframe.py)</span>

<div className="content">

A rolling window of size ‘window’, by default right aligned.

To use grouping as well, first call [`group_by()`](/modeling/bach/api-reference/DataFrame/bach.DataFrame.group_by.mdx) on this frame and call rolling on the result.


### Parameters

* **`window`** (int) – the window size.
* **`min_periods`** (Optional[int]) – the min amount of rows included in the window before an actual value is returned.
* **`center`** (bool) – center the result, or align the result on the right.
* **`closed`** (str) – make the interval closed on the ‘right’, ‘left’, ‘both’ or ‘neither’ endpoints.
Defaults to ‘right’, and the rest is currently unsupported.



### Returns

a new DataFrame object with the [`group_by`](/modeling/bach/api-reference/DataFrame/bach.DataFrame.group_by.mdx) attribute set with a
`bach.partitioning.Window`.



### Return type

[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)


:::note

The `win_type`, `axis` and `method` parameters as supported by pandas, are currently not
implemented.


:::
