# bach.DataFrame.value_counts

<div class="method">
<h2 class="signature-class"><span class="additional-name">DataFrame.</span><span class="name">value_counts(</span><small class="parameter-list"><span class="parameter" id="subset">subset=None</span>, <span class="parameter" id="normalize">normalize=False</span>, <span class="parameter" id="sort">sort=True</span>, <span class="parameter" id="ascending">ascending=False</span></small>)</h2>


<div class="content">

Returns a series containing counts of each unique row in the DataFrame


### Parameters

* **`subset`** (_Optional_[_List_[_str_]_]) – a list of series labels to be used when counting. If subset is not provided and
dataframe has no group_by, all data columns will be used. In case the DataFrame has a group_by,
series in group_by will be added to subset.

* **`normalize`** (_bool) – returns proportions instead of frequencies

* **`sort`** (_bool) – sorts result by frequencies

* **`ascending`** (_bool) – sorts values in ascending order if true.



### Returns

a series containing all counts per unique row.



### Return type

[Series](/docs/modeling/bach/api-reference/Series/bach.Series/#bach.Series)
