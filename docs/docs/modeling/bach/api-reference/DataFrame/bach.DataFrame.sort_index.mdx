# bach.DataFrame.sort_index

<div class="method">
<h2 class="signature-class"><span class="additional-name">DataFrame.</span><span class="name">sort_index(</span><small class="parameter-list"><span class="parameter" id="level">level=None</span>, <span class="parameter" id="ascending">ascending=True</span></small>)</h2>


<div class="content">

Sort dataframe by index levels.


### Parameters

* **`level`** (_Optional_[_Union_[_int_, _List_[_int_]_, _str_, _List_[_str_]_]_]) – int or level name or list of ints or level names.
If not specified, all index series are used

* **`ascending`** (_Union_[_bool_, _List_[_bool_]_]) – Whether to sort ascending (True) or descending (False). If this is a list, then the
`level` must also be a list and `len(ascending) == len(level)`.



### Returns

a new DataFrame with the specified ordering,
otherwise it updates the original and returns None.



### Return type

[bach.dataframe.DataFrame](/docs/modeling/bach/api-reference/DataFrame/bach.DataFrame/#bach.DataFrame)
