<h2 class="signature-class"><span class="additional-name">ObjectivStack.</span><span class="name">get_from_context_with_type_series(</span><small class="parameter-list"><span class="parameter" id="type">type</span>, <span class="parameter" id="key">key</span>, <span class="parameter" id="dtype">dtype='string'</span></small>)</h2>


<div class="content">

Returns the value of `key` from the first context in an Objectiv stack where `_type` matches `type`.


### Parameters

* **`type`** – the _type to search for in the contexts of the stack.

* **`key`** – the value of the key to return of the context with matching type.

* **`dtype`** – the dtype of the series to return.



### Returns

a series of type `dtype`
