# modelhub.ModelHub.to_metabase

<div className="method">

## <span className="additional-name">ModelHub.</span><span className="name">to_metabase</span>

<small className="parameter-list">(<span className="parameter" id="data">data</span>, <span className="parameter" id="model_type">model_type=None</span>, <span className="parameter" id="config">config=None</span></small>)


&#8203;<span className="view-source">[[source]](https://github.com/objectiv/objectiv-analytics/blob/main/modelhub/modelhub/modelhub.py)</span>

<div className="content">

Plot data in `data` to Metabase. If a card already exists, it will be updated. If `data` is a
[`bach.Series`](/modeling/bach/api-reference/Series/index.mdx), it will call [`bach.Series.to_frame()`](/modeling/bach/api-reference/Series/bach.Series.to_frame.mdx).

Default options can be overridden using the config dictionary.


### Parameters

* **`data`** – [`bach.DataFrame`](/modeling/bach/api-reference/DataFrame/index.mdx) or [`bach.Series`](/modeling/bach/api-reference/Series/index.mdx) to push to MetaBase.
* **`model_type`** (Optional[str]) – Preset output to Metabase for a specific model. eg, ‘unique_users’
* **`config`** (Optional[dict]) – Override default config options for the graph to be added/updated in Metabase.
