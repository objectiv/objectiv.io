# modelhub.ModelHub.map

<div className="property">

<h2 className="signature-property"><span className="type-annotation"><em>property</em></span> <span className="additional-name">ModelHub.</span><span className="name">map</span></h2>


<div className="content">

Access map methods from the model hub.

<div className="class">

<h2 className="signature-class"><span className="type-annotation"><em>class</em></span> <span className="name">Map(</span><small className="parameter-list"><span className="parameter" id="mh">mh</span></small>)</h2>


<div className="content">

Methods in this class can be used to map data in a DataFrame with Objectiv data to series values.

Always returns Series with same index as the DataFrame the method is applied to, so the result can be set
as columns to that DataFrame.

<div className="method">

<h2 className="signature-method"><span className="additional-name">Map.</span><span className="name">conversions_counter(</span><small className="parameter-list"><span className="parameter" id="data">data</span>, <span className="parameter" id="name">name</span>, <span className="parameter" id="partition">partition='session_id'</span></small>)</h2>


<div className="content">

Counts the total number of conversions given a partition (ie session_id
or user_id).


### Parameters

* **`name`** – the name of the conversion to label as set in
[`ModelHub.conversion_events`](/docs/modeling/open-model-hub/api-reference/ModelHub/modelhub.ModelHub.conversion-events).


* **`partition`** – the partition over which the number of conversions are counted. Can be any column
of the ObjectivFrame





### Returns

[`bach.SeriesBoolean`](/docs/modeling/bach/api-reference/Series/Boolean) with the same index as `data`.



</div>


</div>

<h2 className="signature-method"><span className="additional-name">Map.</span><span className="name">conversions_in_time(</span><small className="parameter-list"><span className="parameter" id="data">data</span>, <span className="parameter" id="name">name</span>, <span className="parameter" id="partition">partition='session_id'</span></small>)</h2>


<div className="content">

Counts the number of time a user is converted at a moment in time given a partition (ie ‘session_id’
or ‘user_id’).


### Parameters

* **`data`** – [`bach.DataFrame`](/docs/modeling/bach/api-reference/DataFrame) to apply the method on.


* **`name`** – the name of the conversion to label as set in
[`ModelHub.conversion_events`](/docs/modeling/open-model-hub/api-reference/ModelHub/modelhub.ModelHub.conversion-events).


* **`partition`** – the partition over which the number of conversions are counted. Can be any column
in `data`.





### Returns

`bach.SeriesInt64` with the same index as `data`.



</div>


</div>

<div className="method">

<h2 className="signature-method"><span className="additional-name">Map.</span><span className="name">pre_conversion_hit_number(</span><small className="parameter-list"><span className="parameter" id="data">data</span>, <span className="parameter" id="name">name</span>, <span className="parameter" id="partition">partition='session_id'</span></small>)</h2>


<div className="content">

Returns a count backwards from the first conversion, given the partition. I.e. first hit before
converting is 1, second hit before converting 2, etc. Returns None if there are no conversions
in the partition or after the first conversion.


### Parameters

* **`data`** – [`bach.DataFrame`](/docs/modeling/bach/api-reference/DataFrame) to apply the method on.


* **`name`** – the name of the conversion to label as set in
[`ModelHub.conversion_events`](/docs/modeling/open-model-hub/api-reference/ModelHub/modelhub.ModelHub.conversion-events).


* **`partition`** – the partition over which the number of conversions are counted. Can be any column
in `data`.





### Returns

`bach.SeriesInt64` with the same index as `data`.



</div>
