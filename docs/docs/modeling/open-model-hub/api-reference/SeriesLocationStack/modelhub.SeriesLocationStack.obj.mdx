### <span className="name">get_from_context_with_type_series</span>

<small className="parameter-list">(<span className="parameter" id="type">type</span>, <span className="parameter" id="key">key</span>, <span className="parameter" id="dtype">dtype='string'</span>)</small>


<div className="content">

<div id="get-from-context-with-type-series" className="hidden-anchor"></div>

Returns the value of `key` from the first context in an Objectiv stack where `_type` matches `type`.


#### Parameters

* **`type`** – the _type to search for in the contexts of the stack.
* **`key`** – the value of the key to return of the context with matching type.
* **`dtype`** – the dtype of the series to return.



#### Returns

a series of type `dtype`
