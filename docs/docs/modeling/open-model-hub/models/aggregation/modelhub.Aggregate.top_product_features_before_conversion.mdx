# modelhub.Aggregate.top_product_features_before_conversion

Calculates what users did before converting by
combining several models from the model hub.


#### Parameters

* **`data`** ([bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)) – [`bach.DataFrame`](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx) to apply the method on.
* **`name`** (str) – label of the conversion event.
* **`location_stack`** ([SeriesLocationStack](/modeling/open-model-hub/api-reference/SeriesLocationStack/modelhub.SeriesLocationStack.mdx)) – the location stack
  * can be any slice of a [`modelhub.SeriesLocationStack`](/modeling/open-model-hub/api-reference/SeriesLocationStack/modelhub.SeriesLocationStack.mdx) type column
  * if None - the whole location stack is taken.

* **`event_type`** (str) – event type. Must be a valid event_type
(either parent or child).



#### Returns

bach DataFrame with results.



#### Return type

[bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/bach.DataFrame.mdx)
