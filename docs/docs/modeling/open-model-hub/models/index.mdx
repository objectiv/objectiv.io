# Models

The [open model hub](/modeling/open-model-hub/index.mdx) has two main types of functions:
[map](/modeling/open-model-hub/models/Mapping/index.mdx) and [aggregate](/modeling/open-model-hub/models/Aggregation/index.mdx).

* [map](/modeling/open-model-hub/models/Mapping/index.mdx) functions always return a series with the same shape and index as the
[`DataFrame`](/modeling/bach/api-reference/DataFrame/index.mdx) they are applied to. This ensures they can be added as a column to that
[`DataFrame`](/modeling/bach/api-reference/DataFrame/index.mdx). [map](/modeling/open-model-hub/models/Mapping/index.mdx) functions that return
[`DataFrame`](/modeling/bach/api-reference/Series/Boolean/index.mdx) can be used with to filter the data.
* [aggregate](/modeling/open-model-hub/models/Aggregation/index.mdx) functions return aggregated data in some form from the
[`DataFrame`](/modeling/bach/api-reference/DataFrame/index.mdx). Can also be accessed with [`agg`](/modeling/open-model-hub/api-reference/ModelHub/modelhub.ModelHub.agg.mdx).

## Map

| [`is_first_session`](/modeling/open-model-hub/models/Mapping/modelhub.Map.is_first_session.mdx)(data)                            | Labels all hits in a session True if that session is the first session of that user in the data.                                                                                                                       |
| [`is_new_user`](/modeling/open-model-hub/models/Mapping/modelhub.Map.is_new_user.mdx)(data[, time_aggregation])             | Labels all hits True if the user is first seen in the period given `time_aggregation`.                                                                                                                                   |
| [`is_conversion_event`](/modeling/open-model-hub/models/Mapping/modelhub.Map.is_conversion_event.mdx)(data, name)                   | Labels a hit True if it is a conversion event, all other hits are labeled False.                                                                                                                                       |
| [`conversions_counter`](/modeling/open-model-hub/models/Mapping/modelhub.Map.conversions_counter.mdx)(data, name[, partition])      | Counts the total number of conversions given a partition (ie session_id or user_id).                                                                                                                                   |
| [`conversions_in_time`](/modeling/open-model-hub/models/Mapping/modelhub.Map.conversions_in_time.mdx)(data, name[, partition])      | Counts the number of time a user is converted at a moment in time given a partition (ie 'session_id' or 'user_id').                                                                                                    |
| [`pre_conversion_hit_number`](/modeling/open-model-hub/models/Mapping/modelhub.Map.pre_conversion_hit_number.mdx)(data, name[, ...])      | Returns a count backwards from the first conversion, given the partition.                                                                                                                                              |
