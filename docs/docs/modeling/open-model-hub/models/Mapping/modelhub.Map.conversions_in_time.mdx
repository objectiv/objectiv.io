# modelhub.Map.conversions_in_time

<div className="method">

## <span className="additional-name">Map.</span><span className="name">conversions_in_time</span>

<small className="parameter-list">(<span className="parameter" id="data">data</span>, <span className="parameter" id="name">name</span>, <span className="parameter" id="partition">partition='session_id'</span></small>)

<div className="content">

Counts the number of time a user is converted at a moment in time given a partition (ie ‘session_id’
or ‘user_id’).


### Parameters

* **`data`** ([bach.dataframe.DataFrame](/modeling/bach/api-reference/DataFrame/index.mdx)) – [`bach.DataFrame`](/modeling/bach/api-reference/DataFrame/index.mdx) to apply the method on.
* **`name`** (str) – the name of the conversion to label as set in
[`ModelHub.conversion_events`](/modeling/open-model-hub/api-reference/ModelHub/modelhub.ModelHub.conversion_events.mdx).
* **`partition`** (str) – the partition over which the number of conversions are counted. Can be any column
in `data`.



### Returns

`bach.SeriesInt64` with the same index as `data`.



### Return type

bach.series.series_numeric.SeriesInt64
